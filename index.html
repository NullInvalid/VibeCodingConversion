AI-Powered Form Conversion

What This Project Is About

Vibe Coding is a project that uses Cursor AI (an AI coding assistant) to help convert traditional online forms into modern, efficient code much faster than manual conversion.

Instead of manually writing form code, we use AI prompts to generate forms automatically.

How It Works

  1. Read the Instructions - Check Instructions.txt for form requirements
  2. Use AI Prompts - Ask Cursor AI to generate the form based on requirements
  3. Review & Test - Check the AI-generated code and make any needed adjustments
  4. Deploy - Use the converted form

Quick Start Steps

Step 1: Read Requirements

  • Open Instructions.txt to see what form needs to be created
  • Note the form version (V2, V4, V5) and any specific requirements

Step 2: Generate Form with AI

Use Cursor AI with prompts like:

"Read Instructions.txt and create a form based on the requirements."

Step 3: Test the Form

  • Check that all fields are present
  • Verify validation works
  • Test responsive design

Notes

  • For reference fields, you can store fields on a txt file
  • Or you can take a screenshot of the reference
  • PDF files aren't read by Cursor AI
